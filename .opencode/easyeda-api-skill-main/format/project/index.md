# 工程日志格式

- [BLOB 真彩图](/cn/format/project/blob.md) - 真彩图数据格式
- [实例值属性覆盖](/cn/format/project/instance.md) - 实例值属性覆盖格式
- [变体](/cn/format/project/variant.md) - 变体数据格式
- [元件分组](/cn/format/project/group.md) - 元件分组数据格式
