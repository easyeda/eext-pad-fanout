# Вывод площадок

JLCEDA / EasyEDA Pro Плагин вывода площадок PCB через переходные отверстия v2.3.5

## Описание

Автоматическое создание переходных отверстий и трасс для выбранных площадок в PCB-дизайне.

**Основные функции:**

- **Вывод одиночной площадки**: выберите одну площадку и выполните вывод через кнопки направлений или с чередованием длины
- **Пакетный вывод**: выберите несколько площадок рамкой и выполните вывод всех в одном направлении в порядке номеров площадок
- **Вывод кнопками направлений**: UI предоставляет 8 кнопок направлений; все формы площадок поддерживают 8-направленный вывод
- **Вывод курсором**: включите режим вывода курсором и нажмите на пустое место на холсте для автоматического определения направления
- **Поддержка глухих/скрытых переходных отверстий**: автоматическое чтение конфигурации глухих/скрытых отверстий из правил проектирования PCB; поддерживаются сквозные, глухие и скрытые переходные отверстия
- **Следование правилам проектирования**: автоматическое чтение размеров переходных отверстий и ширины трасс из правил проектирования PCB при запуске; поддержка автоопределения единиц mm/mil и переключения
- **Следование правилам цепей**: при переключении площадок автоматически обновляется ширина трассы по умолчанию на основе правил цепи выбранной площадки
- **Чередующийся вывод**: настройка длины чередования для одиночных и пакетных площадок; соседние площадки чередуют разную длину вывода
- **Сворачиваемая панель**: основная панель сворачивается в мини-всплывающее окно с кнопками обновления, переключения вывода курсором и разворачивания
- **Поддержка повёрнутых площадок**: для неповоротных площадок с углом поворота кнопки направлений соответствуют единым направлениям диска
- **Запоминание настроек**: сохраняет последнюю использованную длину трассы

## Использование

1. Запустите плагин из верхнего меню EDA: **Вывод площадок** → **Вывод площадок…**
2. Выберите площадку щелчком или рамкой на холсте PCB
3. Нажмите кнопки направлений в панели UI для вывода; или включите «Вывод курсором» и нажмите на пустое место на холсте
4. Настройте следующие параметры в панели UI:
   - Тип переходного отверстия (сквозное / глухое / скрытое)
   - Внешний диаметр переходного отверстия, диаметр отверстия (mm/mil)
   - Ширина трассы, длина трассы (mm/mil)
   - Длина чередования (для чередующейся длины трасс при выводе)
5. Нажмите «↺ Обновить» для перечитывания текущих правил проектирования PCB
6. Нажмите «−» для сворачивания основной панели; при сворачивании появится мини-всплывающее окно

## Примечания

- При рамочном выборе нескольких площадок вывод курсором определяет направление по ближайшей к курсору площадке

## Сборка

```shell
npm install          # Установка зависимостей
npm run compile      # TypeScript → dist/index.js
npm run build        # Компиляция + упаковка → build/dist/*.eext
npm run lint         # Проверка ESLint
npm run fix          # Автоисправление ESLint
```

Собранный файл `.eext` находится в `build/dist/`. Импортируйте его в менеджере расширений EDA.

## Установка

**EDA Pro V3:** Верхнее меню → Дополнительно → Менеджер расширений… → Импорт
