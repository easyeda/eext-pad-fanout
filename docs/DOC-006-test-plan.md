# DOC-006: 测试计划

## 6.1 测试环境要求

| 项目 | 要求 |
|------|------|
| 操作系统 | Windows 10/11 |
| EasyEDA Pro | >= 3.0.0 |
| Node.js | >= 20.17.0 |
| pro-api-sdk | >= 1.3.0 |
| 测试PCB文件 | 包含至少一个焊盘的PCB文件 |

## 6.2 测试前准备

### 6.2.1 环境检查清单

```
□ EasyEDA Pro 已启动并登录
□ 测试用PCB文件已打开
□ pro-api-sdk 依赖已安装 (npm install)
□ 插件已编译 (npm run build)
□ 插件已安装到EasyEDA Pro
□ 插件已启用
```

### 6.2.2 测试数据准备

| 数据类型 | 要求 |
|----------|------|
| PCB文件 | 包含至少10个不同网络的焊盘 |
| 特殊焊盘 | 包含电源焊盘（GND、VCC等） |
| 密集区域 | 包含焊盘间距较小的区域 |

## 6.6 测试执行检查清单

### 测试前
```
□ 确认EasyEDA Pro正常运行
□ 确认测试PCB文件已打开
□ 确认插件已启用
□ 确认弹窗可以正常显示
```

### 功能验证
```
□ 菜单可点击
□ 弹窗正常显示/隐藏
□ 焊盘可正常选中
□ 预览实时跟随鼠标
□ 过孔创建成功
□ DRC检查生效
```

### 异常处理
```
□ 右键取消正常
□ ESC取消正常
□ 无焊盘区域点击无报错
```

### 测试后
```
□ 清理测试产生的过孔
□ 记录发现的问题
□ 更新测试状态
```
