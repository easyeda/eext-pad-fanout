# DOC-001: 项目概述

## 1.1 项目信息

| 字段 | 内容 |
|------|------|
| 项目名称 | Pad Fanout (单焊盘扇出过孔) |
| 插件类型 | EasyEDA Pro 扩展插件 |
| 核心功能 | 选中单个焊盘，根据鼠标方向进行扇出过孔 |
| 目标用户 | PCB设计工程师 |
| SDK版本 | pro-api-sdk >= 1.3.0 |
| EDA版本 | EasyEDA Pro >= 3.0.0 |

## 1.2 核心价值

- **效率提升**: 一键扇出，无需手动放置过孔
- **规则合规**: 自动遵循PCB设计规则（DRC）
- **视觉直观**: 实时预览，所见即所得

## 1.3 约束条件

- 仅支持 PCB 编辑器环境
- 仅支持单焊盘扇出
- 预览过孔使用虚线样式

## 1.4 术语表

| 术语 | 定义 |
|------|------|
| 扇出 (Fanout) | 将焊盘连接到内层铜皮的过孔和走线 |
| 过孔 (Via) | 连接不同PCB层的金属化孔 |
| 通孔 (Through Via) | 贯穿所有铜层的过孔 |
| 盲孔 (Blind Via) | 从外层到内层的过孔 |
| 埋孔 (Buried Via) | 仅在内层之间的过孔 |
| DRC | Design Rule Check，设计规则检查 |

## 1.5 目标

本项目旨在为EasyEDA Pro提供一个便捷的单焊盘扇出工具，帮助工程师快速完成PCB Fanout操作，提高设计效率。
