# DOC-002: 功能规格说明书

## 2.1 用户操作流程

```
┌─────────────────────────────────────────────────────────────┐
│                      用户操作流程                            │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│   [1] 菜单点击           [2] 焊盘选中        [3] 实时预览    │
│   ┌─────────────┐       ┌─────────────┐      ┌────────────┐│
│   │ PCB菜单栏   │  →    │ 点击焊盘    │  →   │ 移动鼠标   ││
│   │ → 扇出工具  │       │             │      │ 预览跟随   ││
│   └─────────────┘       └─────────────┘      └────────────┘│
│                                                          ↓  │
│   [6] 状态重置        [5] DRC检查        [4] 确认扇出      │
│   ┌─────────────┐       ┌─────────────┐      ┌────────────┐│
│   │ ESC/右键    │  ←    │ 规则验证    │  ←   │ 左键点击   ││
│   │ 取消操作    │       │ 通过/失败   │      │ 创建过孔   ││
│   └─────────────┘       └─────────────┘      └────────────┘│
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

## 2.2 功能列表

| 功能ID | 功能名称 | 优先级 | 描述 |
|--------|----------|--------|------|
| F001 | 菜单注册 | P0 | 在PCB菜单栏注册"单焊盘扇出"菜单项 |
| F002 | 焊盘选中 | P0 | 点击PCB焊盘时选中并记录 |
| F003 | 方向计算 | P0 | 根据鼠标位置计算扇出角度 |
| F004 | 预览生成 | P0 | 鼠标移动时实时显示预览过孔（虚线） |
| F005 | 扇出确认 | P0 | 左键点击确认，创建真实过孔和走线 |
| F006 | 操作取消 | P0 | 右键点击或ESC键取消当前操作 |
| F007 | 预览清除 | P0 | 确认/取消时清除预览过孔 |
| F008 | DRC检查 | P1 | 创建过孔前检查最小间距、线宽规则 |
| F009 | 盲埋孔选择 | P1 | 支持通孔、盲孔1-2、盲孔1-3三种类型 |
| F010 | 电源识别 | P2 | 自动识别电源焊盘网络 |

### 优先级定义

| 优先级 | 说明 |
|--------|------|
| P0 | 核心功能，必须实现 |
| P1 | 重要功能，计划实现 |
| P2 | 增强功能，后续迭代 |

## 2.3 UI规格

### 2.3.1 弹窗规格

| 属性 | 值 |
|------|-----|
| 位置 | 固定右上角（top: 10px, right: 10px） |
| 宽度 | 180px |
| 高度 | 自适应（约100px） |
| 边框圆角 | 4px |
| 阴影 | 0 2px 8px rgba(0,0,0,0.15) |
| 背景色 | #ffffff |
| 边框色 | #cccccc |

### 2.3.2 弹窗内容

```
┌────────────────────────────┐
│  单焊盘扇出                │  ← 标题
├────────────────────────────┤
│  过孔类型:                 │  ← 标签
│  [通孔           ▼]        │  ← 下拉选择
├────────────────────────────┤
│  左键: 确认扇出            │
│  右键/ESC: 取消           │  ← 提示信息
└────────────────────────────┘
```

### 2.3.3 下拉选项

| 值 | 中文 | English | 过孔类型 |
|----|------|---------|----------|
| through | 通孔 | Through | THROUGH |
| blind_1_2 | 盲孔1-2 | Blind 1-2 | BLIND_BURIED |
| blind_1_3 | 盲孔1-3 | Blind 1-3 | BLIND_BURIED |

## 2.4 数据规格

### 2.4.1 扇出距离

| 参数 | 值 | 单位 |
|------|-----|------|
| 默认扇出距离 | 2.54 | mm (100mil) |

### 2.4.2 过孔规格

| 属性 | 值 |
|------|-----|
| 过孔直径 | 继承焊盘属性或默认0.6mm |
| 孔径 | 继承焊盘属性或默认0.3mm |
| 预览样式 | 虚线（dashed） |

### 2.4.3 走线规格

| 属性 | 值 |
|------|-----|
| 走线宽度 | 遵循DRC最小线宽规则 |
| 走线层 | 与焊盘所在层相同 |

## 2.5 用户交互规则

| 操作 | 触发 | 结果 |
|------|------|------|
| 点击菜单 | PCB菜单 → 扇出工具 → 单焊盘扇出 | 启用扇出模式，显示弹窗 |
| 点击焊盘 | 扇出模式下点击焊盘 | 选中焊盘，开始预览 |
| 移动鼠标 | 焊盘选中后移动 | 预览过孔跟随鼠标方向 |
| 左键点击 | 预览状态下左键 | 确认扇出，创建过孔和走线 |
| 右键点击 | 任意状态下右键 | 取消操作，清除预览 |
| ESC按键 | 任意状态下按ESC | 取消操作，关闭弹窗 |

## 2.6 异常处理

| 场景 | 处理方式 |
|------|----------|
| 点击空白处 | 无反应，保持当前状态 |
| 点击非焊盘图元 | 无反应，保持当前状态 |
| DRC检查失败 | 显示警告消息，不创建过孔 |
| 未选中焊盘左键点击 | 无反应 |
