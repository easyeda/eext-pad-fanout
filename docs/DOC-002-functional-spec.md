# DOC-002: 功能规格说明书

## 2.1 用户操作流程

```
┌─────────────────────────────────────────────────────────────┐
│                      用户操作流程                            │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│   [1] 菜单点击         [2] 焊盘选中         [3] 目标点击     │
│   ┌─────────────┐       ┌─────────────┐       ┌──────────┐│
│   │ PCB菜单栏   │  →    │ 点击焊盘    │  →    │ 点击目标 ││
│   │ → 扇出工具  │       │ 记录X点位置 │       │ 记录A点  ││
│   └─────────────┘       └─────────────┘       └──────────┘│
│                                                             │
│   [4] 预览生成         [5] 确认扇出         [6] 状态重置    │
│   ┌─────────────┐       ┌─────────────┐       ┌──────────┐│
│   │ 计算XA方位  │  →    │ 左键点击    │  →    │ 重置状态 ││
│   │ X沿XA方向   │       │ 创建过孔    │       │ 等待下次  ││
│   │ 显示预览    │       │ 走线        │       │          ││
│   └─────────────┘       └─────────────┘       └──────────┘│
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

### 状态机说明

| 状态 | 描述 | 触发条件 | 下一状态 |
|------|------|----------|----------|
| IDLE | 等待选中焊盘 | 默认状态 | PAD_SELECTED |
| PAD_SELECTED | 焊盘已选中，等待目标点 | 点击焊盘 | PREVIEWING |
| PREVIEWING | 显示扇出预览 | 点击目标位置 | IDLE |

### 核心算法

```
已知：
  - X点：被选中焊盘的中心位置 (xi, yj)
  - A点：用户点击的目标位置 (x, y)

计算：
  1. 方位角 θ = atan2(y - yj, x - xi)
  2. 过孔位置 V = X + distance × (cos θ, sin θ)
     其中 distance = 2.54mm (默认扇出距离)

预览：
  - 过孔位于 V 点
  - 走线从 X 到 V
```

## 2.2 功能列表

| 功能ID | 功能名称 | 优先级 | 描述 |
|--------|----------|--------|------|
| F001 | 菜单注册 | P0 | 在PCB菜单栏注册"单焊盘扇出"菜单项 |
| F002 | 状态机管理 | P0 | 管理IDLE/PAD_SELECTED/PREVIEWING三种状态 |
| F003 | 焊盘选中 | P0 | 点击PCB焊盘时选中并记录焊盘位置(X点) |
| F004 | 目标点获取 | P0 | 在PAD_SELECTED状态下获取鼠标点击位置(A点) |
| F005 | 方位计算 | P0 | 根据X点和A点计算扇出方向角度 |
| F006 | 预览生成 | P0 | 在X点沿XA方向显示过孔和走线预览 |
| F007 | 扇出确认 | P0 | 在PREVIEWING状态下点击确认，创建真实过孔和走线 |
| F008 | 操作取消 | P0 | 右键点击或ESC键取消当前操作 |
| F009 | 预览清除 | P0 | 确认/取消时清除预览过孔 |
| F010 | DRC检查 | P1 | 创建过孔前检查最小间距、线宽规则 |
| F011 | 盲埋孔选择 | P1 | 支持通孔、盲孔1-2、盲孔1-3三种类型 |
| F012 | 电源识别 | P2 | 自动识别电源焊盘网络 |

### 优先级定义

| 优先级 | 说明 |
|--------|------|
| P0 | 核心功能，必须实现 |
| P1 | 重要功能，计划实现 |
| P2 | 增强功能，后续迭代 |

## 2.4 数据规格

### 2.4.1 扇出距离

| 参数 | 值 | 单位 |
|------|-----|------|

### 2.4.2 过孔规格

| 属性 | 值 |
|------|-----|
| 过孔直径 | 默认0.6mm |
| 孔径 | 默认0.3mm |

| 属性 | 值 |
|------|-----|

## 2.5 用户交互规则

| 操作 | 触发 | 当前状态 | 结果 |
|------|------|----------|------|
| 点击菜单 | PCB菜单 → 扇出工具 → 单焊盘扇出 | - | 打开弹窗，状态=IDLE |
| 点击焊盘 | 扇出模式下点击焊盘 | IDLE | 记录焊盘位置X，状态=PAD_SELECTED |
| 点击目标 | 焊盘选中后点击任意位置 | PAD_SELECTED | 记录目标A，显示预览，状态=PREVIEWING |
| 点击确认 | 预览状态下左键点击 | PREVIEWING | 创建过孔和走线，重置状态 |
| 右键点击 | 任意状态下右键 | 任意 | 取消操作，清除预览，重置状态 |
| ESC按键 | 任意状态下按ESC | 任意 | 取消操作，关闭弹窗 |

## 2.6 异常处理

| 场景 | 处理方式 |
|------|----------|
| 点击空白处（未选中焊盘） | 无反应，保持IDLE状态 |
| 点击非焊盘图元 | 无反应，保持当前状态 |
| 点击已选中焊盘 | 无反应，保持PAD_SELECTED状态 |
| DRC检查失败 | 显示警告消息，不创建过孔 |
