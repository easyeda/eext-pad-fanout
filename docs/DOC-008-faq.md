# DOC-008: 常见问题FAQ

## 安装与配置

### Q1: 如何安装插件？

**A**:
1. 运行 `npm run build` 编译打包
2. 在EasyEDA Pro中打开扩展管理器
3. 点击"加载扩展"
4. 选择生成的 `.eext` 文件（位于 `build/dist/` 目录）
5. 重启EasyEDA Pro使插件生效

---

### Q2: 插件加载失败怎么办？

**A**: 按以下步骤排查：

1. **检查编译错误**
   ```bash
   npm run lint
   npm run compile
   ```

2. **检查UUID**
   - 打开 `extension.json`
   - 确保 `uuid` 字段有值（打包时会自动生成）

3. **检查控制台**
   - 打开EasyEDA Pro开发者工具
   - 查看控制台错误信息

---

### Q3: 菜单项不显示？

**A**:
1. 确认 `extension.json` 中 `headerMenus` 配置正确
2. 确认 `registerFn` 与导出的函数名一致
3. 重启EasyEDA Pro

---

## 功能使用

### Q4: 点击焊盘没有反应？

**A**:
1. 确认已启用扇出模式（弹窗已显示）
2. 确认点击的是焊盘，不是走线或其他图元
3. 检查焊盘是否在可选择的层上

---

### Q5: 预览过孔为什么是虚线？

**A**: 虚线预览是设计的一部分，用于区分预览状态和真实过孔：
- 预览过孔（虚线）：临时显示，帮助用户确认方向
- 真实过孔（实线）：确认扇出后创建

确认扇出后，预览过孔会被删除。

---

### Q6: 扇出距离可以调整吗？

**A**: 当前版本扇出距离固定为 2.54mm (100mil)。可通过修改代码中的 `DEFAULT_FANOUT_DISTANCE` 常量调整。

---

### Q7: 为什么DRC检查失败后过孔不创建？

**A**: 这是预期行为。DRC（设计规则检查）确保过孔符合PCB设计规范：
- 过孔间距过小
- 违反最小线宽规则

当检查失败时，系统会：
1. 显示警告消息
2. 不创建过孔
3. 保持预览状态，用户可调整方向后重试

---

### Q8: 支持多层PCB的盲埋孔吗？

**A**: 当前版本支持以下过孔类型：

| 类型 | 说明 | 配置 |
|------|------|------|
| 通孔 | 贯穿所有铜层 | through |
| 盲孔1-2 | 顶层到第2层 | blind_1_2 |
| 盲孔1-3 | 顶层到第3层 | blind_1_3 |

更复杂的层配置（如盲孔2-4、埋孔）可后续扩展。

---

### Q9: 如何取消当前操作？

**A**: 有三种方式取消操作：
1. **右键点击** - 取消当前扇出
2. **按ESC键** - 取消当前扇出
3. **再次点击菜单** - 重新启用扇出模式

---

### Q10: 可以连续扇出多个焊盘吗？

**A**: 可以。完成一次扇出后：
1. 点击另一个焊盘即可继续
2. 无需重新启用扇出模式
3. 弹窗保持显示

---

## 技术问题

### Q11: 预览过孔显示位置不对？

**A**: 可能原因：
1. **坐标系问题** - 检查 `calculateAngle` 函数
2. **单位问题** - 确认使用mm而非mil
3. **缓存问题** - 清除预览后重新尝试

---

### Q12: 过孔创建后位置偏移？

**A**: 检查以下内容：
1. `DEFAULT_FANOUT_DISTANCE` 值是否正确
2. `Math.cos/sin` 角度计算是否正确
3. 焊盘中心点获取是否正确

---

### Q13: DRC检查不生效？

**A**:
1. 确认 `eda.pcb_Drc.check()` 调用参数正确
2. 检查返回结果的处理逻辑
3. 查看控制台是否有错误信息

---

### Q14: 如何查看调试日志？

**A**:
1. 打开EasyEDA Pro开发者工具（F12）
2. 切换到Console标签
3. 查看日志输出

或使用：
```typescript
eda.sys_Log.log('Your message', eda.eSYS_LogType.Info);
```

## 其他

**A**:
1. 打开EasyEDA Pro扩展管理器
2. 找到"Pad Fanout"插件
3. 点击"卸载"

---

### Q16: 如何更新插件？

**A**:
1. 修改代码
2. 运行 `npm run build`
3. 在扩展管理器中重新加载 `.eext` 文件
4. 重启EasyEDA Pro

---

### Q17: 插件会影响其他功能吗？

**A**: 不会。该插件：
- 仅在扇出模式下拦截鼠标事件
- 不修改PCB设计数据
- 随时可取消/关闭

## 词汇表

| 术语 | 英文 | 说明 |
|------|------|------|
| 扇出 | Fanout | 将焊盘连接到内层铜皮的过孔和走线 |
| 过孔 | Via | 连接不同PCB层的金属化孔 |
| 通孔 | Through Via | 贯穿所有铜层的过孔 |
| 盲孔 | Blind Via | 从外层到内层的过孔 |
| 埋孔 | Buried Via | 仅在内层之间的过孔 |
| DRC | Design Rule Check | 设计规则检查 |
| 原理图 | Schematic | 电路原理图 |
| PCB | Printed Circuit Board | 印制电路板 |
