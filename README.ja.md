# パッドファンアウト

JLCEDA / EasyEDA Pro PCB パッドファンアウトビアプラグイン v2.3.5

## 機能紹介

PCB設計において、選択したパッドに自動的にファンアウトビアと配線を作成します。

**主な機能：**

- **単一パッドファンアウト**：単一パッドを選択し、方向ボタンまたは千鳥設定でファンアウトを実行
- **一括ファンアウト**：複数パッドをボックス選択し、統一方向で一括ファンアウト（パッド番号順に実行）
- **方向ボタンファンアウト**：UIに8つの方向ボタンを提供、すべてのパッド形状で8方向ファンアウトに対応
- **カーソルファンアウト**：カーソルファンアウトモードを有効にすると、キャンバスの空白部分をクリックして自動的に方向を判定しファンアウト
- **ブラインド/埋め込みビア対応**：PCB設計ルールからブラインド/埋め込みビア設定を自動読み取り、スルーホール/ブラインドビア/埋め込みビアの選択に対応
- **設計ルール追従**：起動時にPCB設計ルールからビアサイズと配線幅を自動読み取りしてデフォルト値に設定、mm/mil単位の自動認識と切り替えに対応
- **ネットルール追従**：パッド切替時、パッドが属するネットのルールに基づいてデフォルト配線幅を自動更新
- **千鳥ファンアウト**：単一パッドおよび一括パッド選択の両方で千鳥長を設定可能、隣接パッドで異なるファンアウト長を交互に使用
- **折りたたみパネル**：メインパネルを折りたたむとミニポップアップが表示され、リフレッシュ、カーソルファンアウト切替、展開ボタンを含む
- **回転パッド対応**：非円形パッドの回転角がある場合、方向ボタンはダイヤルと一致する方向に対応
- **設定記憶**：前回使用した配線長設定を保存

## 使用方法

1. EDAトップメニューの **パッドファンアウト** → **パッドファンアウト…** からプラグインを起動
2. PCBキャンバスでパッドをシングルクリックまたはボックス選択
3. UIパネルの方向ボタンをクリックしてファンアウト実行、または「カーソルファンアウト」を有効にしてキャンバスの空白部分をクリック
4. UIパネルで以下のパラメータを調整可能：
   - ビアタイプ（スルーホール/ブラインドビア/埋め込みビア）
   - ビア外径、穴径（mm/mil）
   - 配線幅、配線長（mm/mil）
   - 千鳥長（ファンアウト時の交互配線長）
5. 「↺ 更新」をクリックして現在のPCB設計ルールを再読み込み
6. 「−」をクリックしてメインパネルを折りたたむと、ミニポップアップが表示されます

## 注意事項

- 複数パッドをボックス選択した場合、カーソルファンアウトはマウスに最も近いパッドを参照原点として方向を判定します

## ビルド

```shell
npm install          # 依存関係のインストール
npm run compile      # TypeScript → dist/index.js にコンパイル
npm run build        # コンパイル + パッケージ → build/dist/*.eext
npm run lint         # ESLintチェック
npm run fix          # ESLint自動修正
```

ビルド生成物 `.eext` ファイルは `build/dist/` にあり、EDA拡張マネージャーでインポートして使用できます。

## インストール

**EDA Pro V3：** トップメニュー → 詳細 → 拡張マネージャー… → インポート
